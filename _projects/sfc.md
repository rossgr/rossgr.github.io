# Aztec, Hilbert and Z-order SFC PE Visualisation Tool


This project was created for my undergraduate final project at Manchester Metropolitan University. The purpose of this project is to compare the usability of the Aztec SFC with other SFC such as the Hilbert and Z-order curve in the context of malware analysis visualisations. 
## Run Locally

Clone the project

```bash
  git clone https://github.com/rossgr/aztec-sfc-visualisations.git
```

Go to the project directory

```bash
  cd aztec-sfc-visualisations/
```

Create samples folder & Place samples in folder

```bash
  mkdir samples
```

Run script to generate images

```bash
  python main.py example.exe
```
